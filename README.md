# static-node-and-express-site

In this project Node.js, Express and Pug.

## Installation

-   In the terminal run:

```
1. npm install
2. npm start
```

## **Static Node and EXpress Site**

### Description

We deploy a Node.js server, for web framework we use Express and html template with Pug.
